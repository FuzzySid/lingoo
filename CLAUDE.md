# Lingoo Project Context

## Project Overview
Trilingual Spanish learning app (Spanish-English-Hindi). 

## Tech Stack
- Frontend: React + Vite + TypeScript + Tailwind CSS
- State: React Hooks
- Storage: Browser LocalStorage
- Design: Pre-generated UI from Stitch (stored in /designs)
